Política de Privacidade

Última atualização: 03/02/2026

Esta Política de Privacidade descreve como o aplicativo Q-Fácil, um app municipal para acesso a informações e serviços do setor público, coleta, utiliza e protege os dados dos usuários.

1. Dados coletados

Podemos coletar os seguintes dados pessoais para funcionamento dos serviços:

  • Nome completo
  • Endereço
  • E-mail
  • Telefone
  • Senha (armazenada de forma segura/criptografada)

2. Dados coletados conforme o serviço utilizado

Agendamento de transporte

  • Comprovante de agendamento médico ou hospitalar
  • Endereço da consulta
  • Dados do paciente necessários para identificação

Reparo de iluminação pública

  • Localização geográfica (quando autorizada pelo usuário)
  • Foto opcional do poste ou do problema relatado

3. Uso das informações

Os dados são utilizados exclusivamente para:

  • Autenticação do usuário
  • Prestação dos serviços públicos solicitados
  • Registro e acompanhamento de solicitações
  • Melhoria da experiência do aplicativo
  • Comunicação com o cidadão quando necessário

4. Compartilhamento de dados

Não vendemos nem comercializamos dados pessoais. As informações podem ser compartilhadas apenas:

  • Com sistemas internos da administração pública responsáveis pelo serviço
  • Quando exigido por obrigação legal
  • Com serviços técnicos essenciais para funcionamento do app (ex: hospedagem, infraestrutura)

5. Localização e permissões do dispositivo

O aplicativo pode solicitar acesso à localização do dispositivo exclusivamente para registrar ocorrências de iluminação pública. O uso é feito apenas com consentimento do usuário.

6. Autenticação

O aplicativo não utiliza login via OAuth ou redes sociais. O acesso é realizado por cadastro próprio do usuário.

7. Dados financeiros

O Q-Fácil não coleta dados financeiros, bancários ou informações de pagamento.

8. Segurança

Adotamos medidas técnicas e organizacionais para proteger os dados contra acessos não autorizados, perda ou uso indevido.

9. Direitos do usuário (LGPD)

Nos termos da Lei Geral de Proteção de Dados (Lei nº 13.709/2018), o usuário pode:

  • Solicitar acesso aos seus dados
  • Corrigir informações incorretas
  • Solicitar exclusão de dados quando aplicável
  • Revogar consentimentos
